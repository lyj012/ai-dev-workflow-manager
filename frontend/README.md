# 前端说明

## 技术栈规划

第一版前端计划使用：

- Vue 3
- TypeScript
- Element Plus

当前阶段只初始化文档和页面规划，暂不创建正式 Vue 项目，暂不编写正式前端业务代码。

## 页面规划

### 任务列表

- 展示任务标题、类型、复杂度、风险标签、状态和更新时间。
- 支持按状态、任务类型、复杂度筛选。
- 支持进入任务详情。

### 任务创建

- 填写任务标题和描述。
- 选择任务类型和复杂度。
- 选择风险标签。
- 创建后进入任务详情或模板匹配流程。

### 任务详情

- 展示任务基础信息。
- 展示匹配的 workflow 模板。
- 展示阶段列表和阶段状态。
- 支持生成阶段指令、记录阶段输出、推进阶段状态。

### 模板管理

- 展示 workflow 模板列表。
- 查看模板详情和阶段配置。
- 维护模板优先级、版本和启用状态。

### 交付记录

- 展示测试清单、测试结果、风险说明和交付总结。
- 支持 Markdown 导出内容预览。

## 暂不创建正式代码

本目录当前仅用于说明前端规划。后续开始编码时，再创建 Vue 3 工程结构，并围绕 MVP 页面逐步实现。
